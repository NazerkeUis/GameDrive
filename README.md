# Mini Escape Drive

## Description
Mini Escape Drive is a 3D arcade game where the player controls a small toy car inside an abandoned playground. The goal is to carefully navigate through obstacles, follow the correct route, and reach the finish.

## Gameplay Features
- Toy car driving mechanics  
- Checkpoint system (4 checkpoints + finish)  
- Sequential route progression  
- Various obstacles across the map  
- Collision-based fail condition  

## Controls
- W / ↑ — Move forward  
- S / ↓ — Move backward  
- A / ← — Turn left  
- D / → — Turn right  

## Win & Lose Conditions

Win:
- Pass all checkpoints in the correct order  
- Reach the finish zone  

Lose:
- Too many collisions with obstacles  

## Technologies
- Unity  
- C#  

## How to Run
1. Download or clone the repository  
2. Open the project in Unity Hub  
3. Load the main scene  
4. Press Play
